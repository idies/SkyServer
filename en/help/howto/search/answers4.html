Practice 8. To find the galaxy with the smallest redshift, use the following query:

    select 
        p.objID, s.z as redshift, p.u, p.r, p.g, p.i, p.z
    from 
        photoObj p, specObj s
    where 
        p.objID = s.bestObjID 
        and s.class = 'galaxy'
        and s.z > 0 
        and p.objID <> 0 
        and s.zWarning = 0
    order by 
        s.z

The query returns ObjID XXX at z = XXX x 10 -5. Using the equation cz = H0d, the galaxy is at a distance of about 800,000 light-years. Comparing the distance to this galaxy to the Andromeda and Whirlpool Galaxies, the distance seems reasonable.

Practice 9. Use the following query:

    select 
        p.run, p.camcol, p.field, count(s.z) as count, avg(s.z) as average
    from 
        photoObj p, specObj s
    where 
        s.bestObjID = p.objID 
        and s.class = 'galaxy'
        and p.run=5112 
        and camcol=1 
    group by 
        p.run,p.camcol,p.field
    order by 
        avg(s.z) desc

The field with the highest average redshift (among fields with more than two measured galaxies) is field 5112-1-220. It includes four spectrally measured galaxies with an average redshift of about 0.3
